# Mohamad Kanso Portfolio

Static GitHub Pages portfolio for Mohamad Kanso.

The site is built as a terminal-style portfolio inspired by eDEX-UI, with public-safe award assets, a downloadable CV, public GitHub project links, and LinkedIn access.

## Local Preview

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.
